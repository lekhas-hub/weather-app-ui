# Weather App UI

## Description
This project is a simple weather app user interface built using HTML and CSS.

## Features
- Search bar for city name
- Weather display card
- Temperature, city name, and humidity display
- Modern gradient background

## Concepts Used
- HTML
- CSS
- Flexbox
- Card design

## How to Run
Open the project in browser to view the weather app design.
